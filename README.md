# Time Series Analysis
### About the Data

In this analysis, the data will be taken from the Washington Post 
[here](https://github.com/washingtonpost/data-police-shootings). Each row
 will consist of: 

- `id`
- `name`
- `date` _YYYY-MM-DD_
- `manner_of_death`
- `armed`
- `age`
- `gender`
- `race`:
  - `W`: White
  - `B`: Black
  - `A`: Asian
  - `N`: Native American
  - `H`: Hispanic
  - `O`: Other
  - `None`: Unknown
- `city`
- `state`
- `signs_of_mental_illness`
- `threat_level`
- `flee`
- `body_camera`
- `latitude` and `longitude`
- `is_geocoding_exact`
